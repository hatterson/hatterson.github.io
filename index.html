Core Finder

This is designed to help you find better spire Cores

Currently it only lists the next 10 cores

This assumes you simply make repeated runs to the spire to select and then stop. If you run fewer or more spires in a run it will change the RNG and invalidate these results.

This also assumes that you will earn no additional spire cores after loading save before starting your next run. Earning more cores will invalidate results
