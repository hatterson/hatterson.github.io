Core Finder

This is designed to help you find better spire Cores

How to use:

Export save from trimps and paste it in the import save box.
If you simply want to see what cores will be coming up next, select the spire you will repeatedly clear and click Search for Cores.
If you wish to limit this list based on required mods, select the "Require certain mods" checkbox and fill in the details. If you enter impossible details, it won't find any.
It will only scan a maximum of 100 runs ahead because if you need more than that you're probably asking for too much

If you wish to engage in RNG seed shenanigans to get a better core earlier (currently hard coded for spire 7/eth cores, dropdown isn't used):
Import save.
Select Require certain mods and select the min values you want on your core
Click "Check RNG seed shenanigans"
Results will tell you how many runs of lower spires you'd need to complete in order for your seed to line up with the seed needed to generate the core you desire

Things to note:

This tool assumes that your save is either from the start of a run (pre-spire 1) or you will portal before earning another spire core.
If you earn another spire core before starting a fresh run or you're currently in the middle of a run, the results are meaningless.
This tool also assume that you will run repeated runs to the spire you select. ie, if you select Spire 4, it assumes that you will only be earning cores by clearing spire 1-4 in runs and will not have any runs in which you fail to clear spire 4 or that you clear spire 5
If you don't do runs like this, results become invalid
